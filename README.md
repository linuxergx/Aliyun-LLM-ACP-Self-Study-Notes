# Aliyun-LLM-ACP-Notes：阿里云大模型 ACP 自学笔记
本仓库包含了针对 **阿里云 ACP（大模型应用开发）** 的学习笔记与第三方题库刷题总结。

### 📌 项目初衷

> 适合人群： 运维工程师、有代码能力但代码能力较差的、着急考证的、有点学不明白的

在备考过程中，我将官方教案中的碎片化知识进行了结构化重组。通过第三方题库刷题一边让AI解答各类知识点来加深ACP课件中的认识。并把常见错题，容易混淆的题整理成笔记的形式进行无偿分析。
